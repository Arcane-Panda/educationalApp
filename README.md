Simple educational app that explains the human body's immune response system.
Specifically, the game teaches about the Complement system and the basic function of the C3 protein.
<img width="1072" height="664" alt="image" src="https://github.com/user-attachments/assets/e6c90413-8908-4a30-bca8-a937529aa4c0" />

Level 1 is a pattern matching game, which represents the chain reaction that first activates the C3 protein:
<img width="1078" height="670" alt="image" src="https://github.com/user-attachments/assets/93d24f8a-1601-4794-9dbf-91a5314adbd8" />

In Level 2, you must help the phagocyte solve a dark maze by following C3a proteins, which lead the way to the site of the infection:
<img width="1064" height="670" alt="image" src="https://github.com/user-attachments/assets/f7273d9e-e638-420f-a917-382d8ae44122" />

In Level 3, you continue to control the phagocyte and swallow up the bacteria that has been immobilized by the C3b proteins:
